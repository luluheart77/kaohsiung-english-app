# LangQuest: 探索高雄 (LangQuest: Kaohsiung)

LangQuest 是一款結合 **高雄城市探索** 與 **聯合國永續發展目標 (SDGs)** 的互動式英語學習應用程式。使用者可以透過地圖導覽高雄各大景點，並與 AI 導覽員進行英語對話，在實際情境中練習語言並深入了解永續議題。

---

## ⏱ 為何現在 (Why Now)

* **政策紅利：** 台灣「2030 雙語政策」目標帶動教育局大幅增加數位學習採購預算，市場需求強勁。
* **技術普及：** ChatGPT 的普及使學生對於使用 AI 學習工具的接受度與意願顯著提升。
* **數位轉型：** 高雄市智慧城市基礎建設完善，開放資料 (Open Data) API 豐富且成熟，為開發應用提供了堅實基礎。

## 📍 為何高雄 (Why Kaohsiung)

* **故事轉型：** 擁有全台最豐富的永續轉型故事，從傳統工業重鎮成功轉型為智慧海洋城市。
* **議題多樣性：** 高雄 38 個行政區跨越都市、鄉村到山地，所涵蓋的 SDG 議題（如生態、工業、族群文化）多樣性無可比擬。
* **政府支持：** 高雄市政府積極推動雙語與永續政策，產官學合作意願高。
* **生態系完整：** 在地創業生態系串聯青創中心、南科加速器及數位發展部南區辦公室，資源整合效率高。

---

## 🌟 核心特色

* **互動式探索地圖**：收錄高雄超過 80 個學習據點，每個地點皆緊扣特定的永續議題。
* **SDG 主題路線**：提供五大精心設計的路線（如：水資源循環、綠能開發、產業轉型、生態多樣性及原民智慧）。
* **AI 智能對話**：導入 Google Gemini / Gemma 模型，提供具備角色背景的 AI 語伴，讓使用者在真實語境下練習英文。
* **永續單字庫**：系統自動記錄對話中的永續專業詞彙，幫助使用者建立專屬學習筆記。
* **多功能輔助工具**：內建中英翻譯、語音轉文字 (STT) 及文字轉語音 (TTS) 功能，降低學習門檻。

## 🛠️ 技術棧

* **前端**：原生 JavaScript (Vanilla JS), HTML5, CSS3。
* **地圖介面**：[Leaflet.js](https://leafletjs.com/)。
* **伺服器端**：[Cloudflare Pages Functions](https://developers.cloudflare.com/pages/)。
* **人工智慧**：Google Gemini API (使用 `gemini-1.5-flash` 與 `gemma-3-27b-it` 模型)。
* **資料庫**：[Supabase](https://supabase.com/)。

## 📂 專案結構

* `public/`：前端靜態資源及 `kaohsiung_tour_data.json` 景點資料庫。
* `functions/api/`：後端 API 邏輯（含 AI 聊天與任務判定）。
* `wrangler.jsonc`：Cloudflare 部署配置。
