# 高雄英語冒險 (Kaohsiung English App)

高雄在地化英語學習應用 - 一個互動式的語言學習平台。

## 專案結構

```
kaohsiung-english-app/
├── public/
│   └── index.html
├── src/
│   ├── App.jsx          # 主應用程式
│   └── index.jsx        # 進入點
├── package.json
└── README.md
```

---

## 技術堆疊

- **React 18** - UI 框架
- **Lucide React** - 圖標
- **GitHub Pages** - 部署平台

---

## 功能

- 🌍 三個高雄地區：美濃、駁二、柴山
- 📍 每個地區有多個景點
- 📚 互動式英語學習課題
- 📊 進度追蹤和成就系統
- 🎯 多種題型：選擇題、配對、語音任務

---

## 備註

- 目前使用 Mock 數據（假資料）
- 語音錄音功能在第 2 階段實裝
- 可以連接 Supabase 數據庫進行完整功能
